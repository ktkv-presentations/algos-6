Ввод и вывод данных

Оператор присваивания

Оператор присваивания в Python обозначается символом = и используется для связывания значения с переменной


a = 5
b = "Привет, мир!"

Левостороннее и правостороннее выражения

Левостороннее выражение

переменная, которой присваивается значение


x = 10 + 5

Левостороннее и правостороннее выражения

Правостороннее выражение

значение или выражение, результат которого присваивается переменной

x = 10 + 5

Множественное присваивание

возможность присваивать значения сразу нескольким переменным в одной строке


a = b = c = 10
					

x, y, z = 1, 2, 3

Множественное присваивание

возможность присваивать значения сразу нескольким переменным в одной строке


a = 10
b = 10
c = 10
					

x = 1
y = 2
z = 3

Правила именования переменных

  • Переменные должны начинаться с буквы или символа подчёркивания _
  • Переменные не могут начинаться с цифры
  • В имени переменной могут быть использованы только буквы, цифры и символ подчёркивания
  • Имена переменных чувствительны к регистру, то есть my_var и my_Var — это разные переменные

user_name = "Alice"
_age = 25
score1 = 100

Ввод данных

процесс получения информации от пользователя, файла или другого источника

Функция input()

позволяет программе взаимодействовать с пользователем, запрашивая ввод данных через консоль

Синтаксис

variable = input("Сообщение для пользователя: ")

где Сообщение для пользователя — это строка, которая выводится на экран перед вводом данных. variable — переменная, которая хранит введённые пользователем данные

Особенности
  • Останавливает выполнение программы и выводит сообщение, ожидая ввода данных от пользователя
  • Возвращает введённые данные в виде строки

Пример использования


name = input("Введите ваше имя: ")
print("Привет,", name)

Обработка ошибок при вводе

Для избежания ошибок при вводе неправильных данных используется конструкция try-except 


try:
		# Код который может вызвать ошибку
except ValueError:
    # Код, который произойдет в случае ошибки
					

import random

answer = random.randint(0, 20)
try:
    guessed_number = int(input("Введите число: "))

    if guessed_number == answer:
        print("Вы угадали!")
    else:
        print("Вы не угадали")
except ValueError:
    print("Ошибка: нужно ввести целое число")

Где ValueError - тип ошибки (исключения) в Python, который возникает, когда функция или операция получает аргумент правильного типа, но с некорректным значением


def my_function():
    try:
        print("Starting the function...")
    except Exception as e:
        print(f"Error: {e}")

    try:
        result = 10 / 0
        print(result)
    except ZeroDivisionError as e:
        print(f"Error: Cannot divide by zero! {e}")

    try:
        my_list = [1, 2, 3]
        print(my_list[5])
    except IndexError as e:
        print(f"Error: Index out of range! {e}")

    try:
        import nonexistent_module
    except ImportError as e:
        print(f"Error: Module not found! {e}")

    try:
        name = "Alice" + 5
    except TypeError as e:
        print(f"Error: Cannot concatenate string and int! {e}")

    try:
        undefined_variable
    except NameError as e:
        print(f"Error: Variable not defined! {e}")

Ввод нескольких значений

Для этого можно воспользоваться методом разделения строки split()

b = int(b)
print(f"Сумма чисел: {a + b}")

Нормализация данных при вводе

процесс приведения данных к определённому стандарту или формату перед их использованием или хранением

" Паша " != "паша"

Обработка ввода строк с использованием методов строк


name = input("Введите ваше имя: ").lower() # "Паша"
print(name) # "паша"

word = input("Введите слово: ").strip() # " Hello world!  "
print(word) # "Hello world!"

Вывод данных

процесс отображения информации, который программа отправляет пользователю или другим программным компонентам

print()

выводит текстовые сообщения или значения переменных в консоль


print("Привет, мир!")
# Привет, мир!

print()


print(значение1, значение2, ..., значениеN, sep=' ', end='\n')

где sep=' ' - разделитель между выводимыми значениями, а end='\n' - символ, который будет добавлен в конце вывода

Usecases


name = "Alice"
age = 25
print("Имя:", name, "Возраст:", age)
# Имя: Alice Возраст: 25
					

print("Python", "быстрый", "язык", sep='-')
# Python-быстрый-язык
					

print("Привет", end=", ")
print("мир!")
# Привет, мир!

Вывод специальных символов

\n
новая строка

print("Привет,\nмир!")
# Привет,
# мир!

Вывод специальных символов

\t
табуляция

print("Привет,\tмир!")
# Привет,    мир!

Вывод специальных символов

\\
обратный слэш

print("Привет,\\мир!")
# Привет,\мир!

Вывод с форматированием чисел


pi = 3.1415926535
print(f"Значение Pi: {pi:.2f}")
# Значение Pi: 3.14

где .2f - указывает на количество после запятой
